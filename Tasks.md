# Tasks

## Goal 

This Python package allows developers to build pure Python APIs without requiring any external dependencies.
JWT support is included by default, so it does not need to be implemented manually.
The generated APIs are automatically safeguarded against the most common SQL injection attacks.

## Main Tasks (features) for version 1.0.0 

- The package can be installed via pip.

- APIs can be created with full CRUD operations.

- Built-in security ensures there are no SQL injection  vulnerabilities.

- Databases can be defined by the user through Models.

- A setup file is automatically generated and utilized.

- APIs can be run directly from the terminal.

- Comprehensive error handling is included.

- JWT support is enabled by default but can be deactivated if not needed.
